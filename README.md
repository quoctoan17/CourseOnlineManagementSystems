# Online Course Management System

Fullstack web application for managing online courses.

## Tech Stack

Frontend:
- React
- Vite
- TailwindCSS

Backend:
- Node.js
- Express
- PostgreSQL

## Features

- Authentication
- Course management
- Role-based access (Admin / Instructor / Student)
- Upload course content

## Run project

Backend
cd Backend
npm install
npm run dev

Frontend
cd Frontend
npm install
npm run dev
